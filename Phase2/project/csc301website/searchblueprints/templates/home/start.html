Search Blueprint Instructions

Please select from the following tabs and see the instructions for the corresponding sections:



  • In "How Search Engine Works" section, you can have a quick overview of how search engine works before breaking it down and diving into the specific part of the search engine.



  • In the Crawlers section, you will be able to learn how search engines finds all the websites. You can learn about a web crawler functions by looking at some of the codes for a simple webcrawler.



  • In the Indexing section, you will be able to learn about the major components of search engine indexting, as well as looking at some of the indexing data structures.



  • In the Compare section, you will be comparing search engines to see how they differ in terms of search results and their ordering.
