How does search engine work?

Search engines gather data using web crawler.

Web crawler (a.k.a web spider) is an Internet bot (software that runs automated scripts on Internet) that returns information by visiting the webpages and following the links that are found within that webpage. The crawling process begins with the list of URLs. When crawler visits each URLs, it adds all the hyperlinks in a webpage into the list of URLs to visit and stores information that will later be processed by the search engine. This process will repeat to update website information, add new URLs or remove URLs that does not exist.

Search engines build index with data

What does the search engine do with the data that was collected by the web crawler? When web crawler finishes crawling a webpage, a search engine will store data in a placed called index. Search engine will store information about the data, and the URL for that webpage.

Search engines use different algorithms to give you back the right results

When user types a query into a search engine, search engine will look through the index to provide the list of best matching webpages. Every search engines have their own algorithms to produce the search results, but PageRank is one of the most well-known algorithm used to imporve the search result. PageRank sorts the result from the index by the popularity of the webpage; the more links that point to a webpage the more useful it is. The other factors of sorting results include the frequency of keywords in a webpage, recently updated webpage and more.
