Creating a crawler

First step is creating a database to store the values and setting it up to connect with whatever language you are working with.
We then create the actual crawler
Here is the algorithm:
- First we add the url to the list of visited pages.
-Then we get the text from the webpage
-We search for links in the text (using regex)
-We then search those links. We can set a depth limit on how deep we can search.
